提示
数据库： 建议为项目创建独立的数据库和用户。
R2 / Turnstile： 安装完成后在后台再配置。
重复执行： 初始化使用 IF NOT EXISTS，多次执行不会破坏表结构。
日志： 可通过 journalctl -u imagebed.service 查看详细错误。
