步骤一：配置数据库
只需填写数据库名称、用户和密码。系统会自动拼接 DSN 并连接 127.0.0.1:5432（sslmode=disable），然后初始化数据表。
步骤二：设置管理员账号
用于登录后台控制台 /admin/。密码会采用 bcrypt 加密存入数据库。
安装成功后，/setup 将自动失效并返回 404，避免重复初始化。
安装小贴士
数据库权限： 建议为该项目创建单独的 PostgreSQL 用户与数据库， 用户需具备在该库内创建表的权限。
R2 与 Turnstile： 为了安装流程更简单，向导只要求数据库。 安装完成后请登录 /admin/ 在控制台中配置 R2 桶与 Cloudflare Turnstile。
重复执行安全： 数据库初始化使用 IF NOT EXISTS， 再次执行不会破坏已有表结构；管理员步骤会对 admins 表进行重建， 以保证结构正确。
遇到错误： 可在服务器上查看 journalctl -u imagebed.service 日志，获取更详细的信息。
