Imagebed 后台管理

管理面板：需要管理员账号登录后才能查看存储桶、配置 Turnstile 等敏感信息。

0. 管理员登录
使用初始化安装时创建的管理员账号登录。登录成功后会解锁下面各个管理区域。
未登录
1. 当前桶列表
登录后可以查看所有已配置的桶，并执行删除等敏感操作。
ID 名称 R2 桶名 配额 (max_bytes) 已用 (current_bytes) 状态 操作

2. 创建新桶
创建时需要管理员登录。最大容量单位为字节，例如 1073741824 ≈ 1 GiB。
3. Turnstile 人机验证配置
在这里配置 Cloudflare Turnstile。配置会写入数据库（turnstile_settings 表）， 不再需要手动编辑配置文件中的 site_key / secret_key。
Site Key（站点密钥）：用于前端页面的 Turnstile 小组件；
Secret Key（秘密密钥）：仅在后端调用 /siteverify 使用，不会在前端暴露。
启用时必须先在下面小组件通过一次验证，后台会使用真实 token 调用 Cloudflare 验证，成功后才允许启用。
